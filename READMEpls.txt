Passo a passo para compilar:

1 - Abra o caminho no CMD ou terminal escolhido
2 - Escreva o código seguindo as regras da linguagem no arquivo "index.txt", ele será reconhecido pelo "reconhecedor.l" 
3 - Foi gerado um arquivo MAKE.file que vai fazer todos os comandos para execução, para rodar apenas é necessário escrever "make"
4 - O código vai ser traduzido em binário para o arquivo "binario.txt".
5 - Este arquivo agora pode ser lido pelo Quartus, não é necessário rodar essa parte thaty :)
6 - A pasta "verilog" contém o esquemático feito em verilog.  
7 - Copie o "binario.txt" e cole ele dentro da pasta verilog

OBS :
O resultado do reconhecimento estará sendo printado no CMD que estará executando o comando "make".
